Features

Missra为您提供了全面的HTML、CSS及JS组件，
它们使用简单，容易定制和扩展。

LESS

Missra 基于LESS & SASS 开发，代码结构清晰简单，易于扩展和维护。

组件

体积小、反应灵敏的响应式组件，使用一致的、无冲突的命名规则。

定制

可根据 Missra 基本的风格及样式，轻松地自定义创建出自己喜欢的主题样式。

响应式

根据不同的屏幕分辨率与上网设备，UIKit 会自动做出响应，提供一致的体验。

Community

GitHub

在GitHub上“clone”Missra，并发送给Missra一个pull请求，您可以提交一个错误或提出一个新的问题。

WeChat

关注Missra的微信公众号 可以第一时间接收相关资讯和更新，欢迎反馈和提问。

开放源码

Missra 是一款开源的前端UI界面框架，
你可以无任何限制的使用 Missra 来创建自己的风格。

MIT 许可协议

Missra 是基于MIT许可协议的开源项目。它完全免费，你可以无任何限制的使用、复制、合并、发布及分发这个框架。
